┌──────────────────────────────────────────────────────────────────────────────┐
│                      🎓  INFORMATIONS ACADÉMIQUES                            │
└──────────────────────────────────────────────────────────────────────────────┘

  Établissement    : ESPRIT
  Niveau           : 4ème année
  Année académique : 2025-2026
  Type de projet   : Projet Intégré (PI)
  Domaine          : Fintech / Blockchain
  Sujet            : Microfinance Décentralisée

  Description du projet :
  ─────────────────────────────────────────────────────────────────────────────
  EQUA est un écosystème de microfinance décentralisé qui tokenise l'argent
  et les actifs pour offrir des prêts, de l'épargne, des paiements, des
  investissements et des transferts P2P instantanés, au-delà de la banque
  traditionnelle.

  Problème résolu :
  ─────────────────────────────────────────────────────────────────────────────
  • 37% seulement de la population tunisienne a accès aux services bancaires
  • 2.5-3.5 millions d'adultes exclus du système financier formel
  • Les cryptomonnaies existantes ne répondent pas aux besoins de microfinance

  Solution EQUA :
  ─────────────────────────────────────────────────────────────────────────────
  • Token EQUA pegged 1:1 avec le TND
  • Partenariat avec la BCT (Banque Centrale de Tunisie)
  • Conformité réglementaire complète
  • Interface accessible et inclusive
  • Frais de transaction quasi-nuls

┌──────────────────────────────────────────────────────────────────────────────┐
│                      🐛  DÉPANNAGE RAPIDE                                    │
└──────────────────────────────────────────────────────────────────────────────┘

  Problème : "npm: command not found"
  ──────────────────────────────────────────────────────────────────────────
  Solution : Installez Node.js depuis https://nodejs.org

  Problème : Port 4200 déjà utilisé
  ──────────────────────────────────────────────────────────────────────────
  Solution : ng serve --port 4300

  Problème : "ng: command not found"
  ──────────────────────────────────────────────────────────────────────────
  Solution : npm install -g @angular/cli

  Problème : Erreurs de compilation TypeScript
  ──────────────────────────────────────────────────────────────────────────
  Solution : Remove-Item -Recurse node_modules && npm install

  Problème : Le canvas 3D ne s'affiche pas
  ──────────────────────────────────────────────────────────────────────────
  Solution : Vérifiez la console (F12) pour les erreurs WebGL
            Assurez-vous que l'accélération matérielle est activée

  Problème : Les animations ne fonctionnent pas
  ──────────────────────────────────────────────────────────────────────────
  Solution : Vérifiez que GSAP est correctement installé
            Regardez la console pour les erreurs GSAP

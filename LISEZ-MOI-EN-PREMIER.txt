                        🔗 BACKEND (Optionnel)
═══════════════════════════════════════════════════════════════════════════════

L'application fonctionne avec des données de démonstration.

Pour connecter un backend Spring Boot :
1. Consultez BACKEND-GUIDE.md
2. Créez le backend dans un dossier séparé
3. Lancez-le sur le port 8080

API attendue :
  GET /api/projects          → Liste des projets
  GET /api/projects/{id}     → Détail d'un projet

═══════════════════════════════════════════════════════════════════════════════
                        🐛 PROBLÈMES COURANTS
═══════════════════════════════════════════════════════════════════════════════

❌ "npm: command not found"
   → Installez Node.js : https://nodejs.org

❌ Port 4200 déjà utilisé
   → Lancez sur un autre port : ng serve --port 4300

❌ "ng: command not found"
   → Installez Angular CLI : npm install -g @angular/cli

❌ Erreurs de compilation
   → Réinstallez : Remove-Item -Recurse node_modules && npm install

   → Éditez src/environments/environment.ts
